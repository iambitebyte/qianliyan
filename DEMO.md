# Qianliyan 新功能演示

## 快速开始

### 1. 启动应用
```bash
cd /home/chenshuyang/project/logs_watcher
npm run dev
```

应用将在 http://localhost:12306 启动

### 2. 访问主页
打开浏览器访问 http://localhost:12306

## 功能演示

### 美化的主页

#### 新特性
- **渐变背景**: 从灰色到浅灰色的渐变效果
- **品牌标识**: 带有 ShieldCheck 图标的 Logo
- **卡片布局**: 现代化的卡片设计
- **加载动画**: 旋转加载指示器
- **响应式**: 适配手机、平板和桌面

#### 操作步骤
1. 查看页面顶部的 Qianliyan Logo 和标题
2. 在"添加监控文件夹"卡片中输入路径（如：`/var/log`）
3. 点击"添加"按钮
4. 在"监控文件夹列表"中查看添加的文件夹
5. 点击文件夹的"打开"按钮查看日志

### 日志查看页面

#### 新特性

##### 1. 全屏布局
- 固定顶部导航栏
- 左侧文件列表
- 右侧日志内容
- 自适应屏幕高度

##### 2. 计时器功能
- 实时显示距离上次更新的时间
- 每秒自动更新
- 友好的时间格式（5秒前、2分钟前、1小时前）
- 仅在日志更新时重置

##### 3. 自动滚动
- 日志更新时自动滚动到底部
- 用户手动滚动时暂停自动滚动 2 秒
- 优雅的滚动条样式

##### 4. 灵活的刷新控制
- 默认 10 秒刷新
- 可选择：1s、2s、5s、10s、30s、手动
- 实时显示当前刷新间隔
- 手动模式提供刷新按钮

#### 操作步骤

1. **选择文件**
   - 点击左侧列表中的文件
   - 文件名高亮显示
   - 右侧显示日志内容

2. **设置刷新间隔**
   - 在顶部右侧选择刷新时间
   - 默认为 10 秒
   - 日志内容将自动更新

3. **查看计时器**
   - 顶部右侧显示"最后更新"时间
   - 时间格式：5秒前、2分钟前等
   - 每秒自动更新

4. **自动滚动**
   - 新日志内容自动滚动到底部
   - 手动滚动时会暂停自动滚动
   - 停止滚动 2 秒后恢复自动滚动

5. **手动刷新**
   - 选择"手动"模式
   - 点击"刷新"按钮
   - 立即获取最新日志内容

## 常见场景

### 场景 1：监控系统日志
1. 添加 `/var/log/syslog` 文件夹
2. 选择 `syslog` 文件
3. 设置刷新间隔为 5s
4. 实时查看系统日志更新

### 场景 2：查看应用日志
1. 添加应用日志文件夹
2. 选择 `application.log` 文件
3. 设置刷新间隔为 30s
4. 查看应用运行日志

### 场景 3：调试模式
1. 添加调试日志文件夹
2. 选择 `debug.log` 文件
3. 设置刷新间隔为 1s
4. 实时查看调试信息

### 场景 4：查看历史日志
1. 选择历史日志文件
2. 设置刷新间隔为"手动"
3. 手动点击刷新按钮
4. 按需更新日志内容

## 快捷操作

### 主页快捷键
- `Enter` - 添加文件夹（在输入框聚焦时）
- 点击文件夹 - 打开文件夹

### 日志页面快捷键
- 点击文件 - 选择并查看
- 下拉菜单 - 选择刷新间隔
- 刷新按钮 - 手动刷新（手动模式）

## 故障排除

### 页面显示异常
1. 清除浏览器缓存
2. 刷新页面
3. 检查网络连接

### 日志不更新
1. 检查刷新间隔设置
2. 确认文件路径正确
3. 检查文件权限

### 滚动问题
1. 手动滚动后等待 2 秒
2. 重新选择文件
3. 刷新页面
