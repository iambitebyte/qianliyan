## 功能演示

### 主页功能
1. 添加文件夹 - 输入路径并点击添加
2. 查看文件夹列表 - 显示所有监控的文件夹
3. 打开文件夹 - 点击文件夹进入日志查看页面
4. 删除文件夹 - 点击删除按钮移除文件夹

### 日志查看页面功能
1. 选择文件 - 从左侧列表选择要查看的日志文件
2. 设置刷新间隔 - 从下拉菜单选择刷新频率
3. 实时更新 - 日志内容自动刷新
4. 计时器显示 - 显示距离上次更新的时间
5. 自动滚动 - 新日志自动滚动到底部
6. 手动刷新 - 手动模式下可点击刷新按钮

## 使用说明

### 启动应用
```bash
cd /home/chenshuyang/project/logs_watcher
npm run dev
```

### 访问地址
```
http://localhost:12306
```

### 添加监控文件夹
1. 在主页输入文件夹路径
2. 点击"添加"按钮
3. 文件夹将添加到列表中

### 查看日志
1. 点击文件夹进入日志查看页面
2. 从左侧选择要查看的文件
3. 日志内容将自动显示
4. 调整刷新间隔以满足需求

## 配置说明

### 默认刷新间隔
当前设置为 10 秒，可在代码中修改：

```tsx
const [refreshInterval, setRefreshInterval] = useState(10);
```

### 自动滚动
日志内容更新时会自动滚动到底部，用户手动滚动时会暂停自动滚动 2 秒。

### 计时器
计时器在日志更新时重置为 0，然后每秒递增，显示友好时间格式。
